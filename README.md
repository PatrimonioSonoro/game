# 🎵 Patrimonio Sonoro - Juego Musical Interactivo

Un juego educativo interactivo diseñado para promover y enseñar sobre la rica tradición musical colombiana, especialmente preparado para presentaciones internacionales como la próxima exhibición en Japón.

## 🌟 Características Principales

### 🎮 Modos de Juego
- **Reconocimiento de Instrumentos**: Identifica instrumentos tradicionales colombianos
- **Identificación de Géneros**: Aprende sobre diferentes géneros musicales regionales
- **Desafío de Ritmo**: Reconoce patrones rítmicos característicos
- **Cultura Musical**: Preguntas sobre historia y tradiciones musicales

### 🎯 Niveles de Dificultad
- **Fácil**: Perfecto para principiantes (5 preguntas, 30s por pregunta)
- **Intermedio**: Para conocedores básicos (8 preguntas, 20s por pregunta)
- **Difícil**: Desafío para expertos (10 preguntas, 15s por pregunta)

### 🏆 Sistema de Logros
- Primeros Pasos
- Maestro del Ritmo
- Experto en Instrumentos
- Erudito Cultural
- Maestro de Géneros
- Puntuación Perfecta

## 🎼 Contenido Musical

### Géneros Incluidos
- **Vallenato** (Costa Caribe) - Patrimonio de la Humanidad UNESCO
- **Cumbia** (Costa Caribe) - Fusión cultural icónica
- **Bambuco** (Región Andina) - Aire tradicional colombiano
- **Joropo** (Llanos Orientales) - Música llanera virtuosa
- **Porro** (Costa Caribe) - Ritmo festivo de bandas
- **Currulao** (Costa Pacífica) - Herencia afrodescendiente
- **Mapalé** (Costa Caribe) - Danza acrobática tradicional
- **Bullerengue** (Costa Caribe) - Expresión femenina ancestral
- **Rajaleña** (Huila/Tolima) - Aire sureño festivo

### Instrumentos Tradicionales
- **Acordeón** - Alma del vallenato
- **Guitarra** - Base de la música andina
- **Tambora** - Corazón rítmico costeño
- **Tiple** - Instrumento nacional colombiano
- **Marimba** - Xilófono del Pacífico
- **Arpa Llanera** - Rey del joropo
- **Guacharaca** - Idiófono único del Caribe
- **Cununos** - Tambores sagrados del Pacífico
- **Bandola** - Melodía de cuerdas pulsadas

## 🚀 Instalación y Uso

### Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a internet (para recursos multimedia)
- Altavoces o auriculares recomendados

### Estructura del Proyecto
```
game_patrimonio_sonoro/
├── html/
│   └── index.html          # Página principal
├── css/
│   └── styles.css          # Estilos y diseño responsivo
├── js/
│   ├── data.js            # Base de datos musical
│   ├── game.js            # Lógica del juego
│   ├── ui.js              # Manejo de interfaz
│   └── main.js            # Aplicación principal
├── assets/
│   ├── audio/             # Muestras musicales
│   ├── images/            # Imágenes de instrumentos y géneros
│   └── sounds/            # Efectos de sonido
└── README.md
```

### Cómo Ejecutar
1. Descarga todos los archivos del proyecto
2. Abre `html/index.html` en tu navegador web
3. ¡Comienza a explorar la música colombiana!

## 🔧 Desarrollo y Personalización

### Agregar Nuevo Contenido
1. **Géneros**: Editar `js/data.js` en la sección `genres`
2. **Instrumentos**: Agregar en la sección `instruments`
3. **Preguntas**: Expandir `culturalQuestions`
4. **Audio**: Colocar archivos en `assets/audio/`
5. **Imágenes**: Agregar a `assets/images/`

### Personalización de Estilos
- Modificar variables CSS en `:root` para cambiar colores
- Ajustar `styles.css` para cambios de diseño
- Personalizar animaciones y transiciones
- Adaptar tipografía y espaciado
